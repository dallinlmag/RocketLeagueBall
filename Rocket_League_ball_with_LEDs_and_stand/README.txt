# Summary

two-part Rocket League ball with holes to insert typical 5mm LEDs. I used 4 LEDs so I could use the bottom hole for a 5mm toggle switch. For power I used a 9V battery.

update: I added some struts to the bottom cover of the stand, so that it holds up the inlay. This removes the need to glue the inlay to the top cover.
